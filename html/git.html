Git

  • git init – создаёт новый репозиторий Git в текущей папке.
  • git clone – копирует существующий репозиторий с сервера на ваш компьютер.
  • git add – добавляет изменения в индекс (готовит к сохранению).
  • git commit – сохраняет изменения в локальном репозитории с описанием.
  • git push – отправляет локальные коммиты в удалённый репозиторий (например, на GitHub).
